# Automated mac setup
---

Prerequisites

- Install Xcode and the Xcode command line tools from the terminal

Installation

Open Terminal and type in the following command:

```
curl -L https://raw.githubusercontent.com/mayank-patel/mac-setup/master/start.sh | sh
```
